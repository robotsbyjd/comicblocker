Comic Blocker

Block your comic before you draw it.

A layout tool for comics writers. Block your panels visually, write your panel-by-panel script underneath, export the whole thing as PDF or compile a clean text script.

Two Modes

  • The Incredible Tutorial — open New / Tutorial to load a six-page guided comic that teaches the tool using the tool.
  • Page mode — work on one page at a time with full controls. The visual layout sits at top; the panel-by-panel script lives below the page.
  • Thumbnail mode — see all your pages at once. Block multiple pages quickly with one-click templates. Drag to nudge panels. Dbl-click a page to jump back to Page mode.

Adding Elements

  • + Panel — the boxes themselves. The visual frames.
  • + Speech / Thought — dialogue bubbles. Add a tail from the properties panel after selecting one.
  • + Caption — narrator boxes ("Meanwhile...", "Three years later...").
  • + SFX — sound effect lettering (BOOM, KRAK, etc).
  • + Image — drop in reference art or photo studies.
  • Panel templates — the dropdown drops a pre-made grid (splash, 6-panel, 9-grid Watchmen-style, etc).

Manipulating

  • Click any element to select it.
  • Drag to move. Drag the blue corner handles to resize.
  • Drag the green handle (above the selection) to rotate.
  • Dbl-click any text element to open a larger editor for bubbles, captions, and SFX.
  • Dbl-click a panel to jump to its script entry below.
  • Press Del or Backspace to remove a selected element.
  • Use Ctrl/Cmd + Z for undo and Ctrl/Cmd + Shift + Z for redo.

Writing the Script

Below each page, every panel gets a numbered script entry. Write what's happening in that panel — camera angle, action, mood. Plus a "Page-Level Direction" field at the top for stuff that applies to the whole page (lighting, motifs, etc).

Click the yellow PANEL # badge in any script entry to highlight that panel on the page above.

Pages

  • + Page / − Page — add or remove pages. Default is 24 (single floppy), can grow up to 200 (full graphic novel). Removing a populated page asks for confirmation.
  • Dup — copy the current page (and its script) to a new page after it. Great for keeping a consistent grid.
  • Spread — toggle two-page facing-spread view (read-only on the right; edit on the left).

Saving & Exporting

  • Save — downloads your project as a .json file. Keep it on your computer like any other document.
  • Load — loads a previously saved .json back into the tool.
  • Autosave recovery — the browser keeps a local recovery copy while you work. Your manual .json is still the source of truth.
  • PDF — exports all pages as a multi-page PDF. Each page in the PDF includes the visual at top and the full script underneath. Send to your artist or print for reference.
  • Compile script — generates a clean text script (Marvel-style) of the whole comic and downloads it as a .txt. Includes all panel descriptions plus dialogue/captions/SFX. This is what you'd hand to a collaborator if they only need the words.
